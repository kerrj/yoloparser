Boxes Around Robots

Boxes around robots is a project collecting training data for detecting robots and common game elements in FTC games. Its goal is to amass thousands of images with boxes drawn around game elements and robots for training object detectors. It is open source at GitHub , where all code for the website resides, as well as training images and formatted annotations.

Why do you need so much data?

Modern image processing techniques like neural networks require vast amount of training data to learn to accomplish a task. Popular research data-bases like ImageNet have millions of images categorized into classes for this purpose. To use neural nets to detect FTC robots, we need thousands of images with robots marked in them. This website is a tool for crowdsourcing that task. Your help is much appreciated!

Instructions

1. Click and drag to draw a box as accurately as possible around an object, then press ENTER to confirm.

2. Type in the name of the object, then press ENTER to confirm.

3. Repeat for all objects in the scene.

4. Press the "S" key to save the annotations.

5. If needed, press ESCAPE to delete the most recent box
